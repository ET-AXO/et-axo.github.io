# Site deco de Noel

Ceci est un site internet qui répertorie les maisons décorées pour Noël sur une carte.

# Installation

1. **Installer** Visual Studio Code.

    https://code.visualstudio.com/Download

2. **Télécharger** le code.

    Aller sur la page du projet, cliquer sur ```<> Code``` et copier/coller l'URL du projet dans VSCode.

3. **Modifier** le code.

4. Le **Versionner** sur github.

    ```
    git add . / git add [nom_fichier]
    git commit -m "message-personnalisé"
    git push origin [nom-branch]
    ```

## Extensions

- Live Server (Permet de visualiser en temps réel le rendu du développement).

## Command Git

- Ajout de branche
```git branch [nom-branche]```

- Changement de branche
```git checkout [nom-branche]```

- Fusion de branche
```git merge [nom-branche à fusionner]```
